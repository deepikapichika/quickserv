# QuickServ - Service Marketplace Platform

QuickServ is a Spring Boot-based service marketplace platform that connects customers, service providers, and administrators through role-based workflows for service discovery, booking, and management.

## 🚀 Project Status
- Core features are implemented and available for local review
- Authentication, role-based dashboards, service management, and booking-related workflows are included
- Live deployment is currently in progress, but the project can be run locally using the setup instructions below

## ✨ Features
- User Authentication for Customer, Provider, and Admin roles
- Secure password encryption using BCrypt
- Role-based dashboards and access control
- Service category and provider management
- Service browsing, search, and filtering
- Booking workflow for customers
- Provider dashboard for managing services and bookings
- Admin dashboard for platform management
- MySQL database integration

## 🛠️ Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap

## 📌 Modules Completed
- Module 1: Authentication System
- Module 2: Service Management
- Module 3: Provider Registration and Management
- Module 4: Booking System
- Module 5: Admin Dashboard
- Module 6: Reviews and Ratings

## 🏗️ Project Highlights
- Clean layered architecture with Controller, Service, and Repository layers
- Secure login and role-based routing
- Separate workflows for customers, providers, and admins
- Responsive UI built with Thymeleaf and Bootstrap
- Organized codebase suitable for real-world service marketplace use


## ⚙️ How to Run Locally
### Prerequisites
- Java 17
- Maven
- MySQL

### Setup Steps
1. Clone the repository
2. Create a MySQL database
3. Update `src/main/resources/application.properties` with your database credentials
4. Run the application using Maven or your IDE

Login Roles
Customer: browse and book services
Provider: manage services and bookings
Admin: manage users, categories, and platform data
